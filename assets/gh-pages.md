## Maven Repository

### For release

#### Maven

```xml
<repository>
    <id>okocraft-box-repo</id>
    <url>https://okocraft.github.io/Box/maven/</url>
</repository>
```

#### Gradle

```
repositories {
    maven {
        url 'https://okocraft.github.io/Box/maven/'
    }
}
```
